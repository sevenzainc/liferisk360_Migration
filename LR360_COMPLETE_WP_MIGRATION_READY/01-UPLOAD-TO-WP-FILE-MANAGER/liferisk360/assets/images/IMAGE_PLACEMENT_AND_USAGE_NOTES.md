## Safe-use rules for LifeRisk360

- Use images to show youth strength, confidence, direction, and support.
- Do not use images to imply that any person shown has experienced risk, harm, poor decisions, or personal challenges.
- Do not place identifiable people beside sensitive claims such as drug use, exploitation, gangs, sexual risk, or crisis wording.
- Use risk-category images as general atmosphere or abstract support only.
- Avoid using images in a way that implies endorsement by the people shown.

## WordPress/Kadence note

The images are stored in `assets/images/` for static preview. In WordPress production, upload the images to the Media Library and replace local paths with WordPress media URLs if needed.
