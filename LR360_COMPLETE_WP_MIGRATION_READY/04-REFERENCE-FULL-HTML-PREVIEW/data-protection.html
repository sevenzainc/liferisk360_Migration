Data Protection

LifeRisk360 protects personal information through governance, technical controls, user roles, and defined retention practices.

Youth-ready design

Protecting youth information responsibly

Data protection limits exposure, supports consent, and keeps youth information inside approved access boundaries.

Data Minimization

Only collect information needed for the program purpose.

Purpose Limitation

Explain why data is collected and how it will be used.

Access Control

Restrict access by role and responsibility.

Retention

Define how long pilot data and reports are retained.

Auditability

Log privileged actions, exports, changes, and administrative reviews.

Keep data use limited, clear, and controlled

Use responsible data practices to protect youth information and support lawful program operations.
